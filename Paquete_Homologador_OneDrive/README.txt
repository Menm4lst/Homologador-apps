📦 HOMOLOGADOR - INSTRUCCIONES DE INSTALACIÓN
===============================================

🎯 QUÉ CONTIENE ESTE PAQUETE:
• Homologador.exe - El programa compilado
• config_onedrive.json - Configuración para OneDrive
• INSTALAR.bat - Instalador automático
• README.txt - Este archivo

🚀 INSTALACIÓN RÁPIDA:
1. Clic derecho en "INSTALAR.bat" → "Ejecutar como administrador"
2. Seguir las instrucciones en pantalla
3. ¡Listo! El programa estará en tu escritorio

📋 QUÉ HACE EL INSTALADOR:
✅ Crea carpeta OneDrive\HomologadorApp
✅ Copia el programa al escritorio
✅ Configura acceso directo
✅ Prepara directorios de backup

⚠️ CONFIGURACIÓN ONEDRIVE (MUY IMPORTANTE):
1. La persona que compila debe COMPARTIR la carpeta 
   "OneDrive\HomologadorApp" contigo
2. Acepta la invitación de OneDrive
3. Asegúrate de que la carpeta se sincronice en tu PC
4. Verifica que veas la misma carpeta en ambos equipos

🔄 CÓMO MANTENER DATOS SINCRONIZADOS:
• Los datos se guardan automáticamente en OneDrive
• Cuando uno guarda cambios, OneDrive los sincroniza
• El otro usuario verá los cambios después de la sincronización
• NO usar el programa al mismo tiempo (puede causar conflictos)

🔐 CREDENCIALES DE ACCESO:
Usuario: admin
Contraseña: admin123

💾 BACKUPS AUTOMÁTICOS:
• El sistema hace copias de seguridad cada 24 horas
• Los backups se guardan en OneDrive\HomologadorApp\backups
• Retiene backups por 30 días automáticamente

🆘 SOLUCIÓN DE PROBLEMAS:

❓ "No veo los datos del otro usuario"
→ Verificar que OneDrive esté sincronizado (icono verde)
→ Reiniciar el programa
→ Verificar permisos de carpeta compartida

❓ "Error al abrir la base de datos"
→ Cerrar el programa en ambos equipos
→ Esperar a que OneDrive sincronice
→ Abrir solo en un equipo a la vez

❓ "El programa no inicia"
→ Verificar que la carpeta OneDrive\HomologadorApp exista
→ Ejecutar como administrador
→ Revisar que config.json esté junto al .exe

🔧 INSTALACIÓN MANUAL (si el .bat no funciona):
1. Crear carpeta: C:\Users\[TuUsuario]\OneDrive\HomologadorApp
2. Copiar Homologador.exe al escritorio
3. Copiar config_onedrive.json al escritorio (renombrar a config.json)
4. Crear acceso directo del .exe

📞 CONTACTO:
Si tienes problemas, contacta a quien te compartió el programa.
